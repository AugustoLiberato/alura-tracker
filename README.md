# alura-tracker
Durante o curso da alura sobre framework Vue 3, fiz esse projeto que utiliza um temporizador para a realização de projetos.

Esse foi um dos vários cursos da Alura que fiz assim que entrei no estágio, para me preparar para as demandas que me foram atribuídas. 

Esse projeto utiliza o json-server que roda no Node.js, json-server faz as operações de CRUD no arquivo db.json

O projeto do vercel está utilizando a versão grátis do My JSON Server, que ainda esta em beta portanto alguns contratempos, como lentidão de respota, acabam acontecendo mesmo

### Instalar todas as dependências 
npm install

### Rodar o Backend
npm run backend

### Em outro terminal rodar o frontend
npm run serve
