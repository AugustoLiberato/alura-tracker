# alura-tracker

## Clonar o repositório
git clone [url-do-repositorio]

## Entrar na pasta
cd alura-tracker

## Instalar todas as dependências (incluindo json-server)
npm install

## Rodar o backend
npm run backend

## Em outro terminal, rodar o frontend
npm run serve
